# Videos for the love story

Add romantic videos, circles, or memories using these names so the cinematic gallery finds them automatically:

- `video-1.mp4` … `video-4.mp4`
- `circle-1.mp4`, `circle-2.mp4`
- `memory-1.mp4`, `memory-2.mp4`

Videos are muted, looped, and autoplay when visible on screen.

# Images for the love story

Replace the placeholder SVG files or add your own photos using these names so the gallery finds them automatically:

- `photo-1.jpg` … `photo-8.jpg`
- `memory-1.jpg`, `memory-2.jpg`
- `together-1.jpg`, `together-2.jpg`

You can also edit `mediaConfig` in `script.js` if you want different file names.
